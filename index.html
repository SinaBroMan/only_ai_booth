✨ Gemini (Canvas)

Dash Board

복잡한 기획안과 현황판을 순식간에. Canvas 기능을 활용해 인터랙션 대시보드를 만들어보세요. 미리보기 기반으로 바로 구조를 확인할 수 있습니다.

1. "도구" 클릭 → 2. "Canvas" 선택
( ※ 구글 로그인 필요)
TRY THIS PROMPT
단일 HTML로 브라우저에서 바로 작동하는 모던한 '건설 현장 관리 대시보드'를 만들어주세요.

화면에는 KPI 카드 4개(공정률, 무재해, 인원, 예산), 차트 2개(주간 공정, 직종별 인원), 안전 체크리스트, 실시간 이슈 테이블이 모두 들어가야 합니다.

반응형 디자인을 적용하고, 가상의 더미 데이터와 인터랙션(체크박스, 차트 렌더링)을 포함해 완벽한 코드로 작성해 주세요.
(※ 모든 UI는 기기 화면 비율에 최적화하여 코딩)
